## 🎯 앱 소개

**따당(Ddadang)**은 아마추어 골퍼를 위한 **올인원 라운딩 앱**입니다.

### ✨ 주요 특징
- 🗓️ **골프 약속 관리**: 일정 조율부터 동반자 초대까지
- 💬 **실시간 채팅**: 약속별 그룹 채팅
- ⚡ **빠른 스코어 입력**: 직관적인 버튼 입력 방식
- 💰 **자동 정산**: 복잡한 골프 정산을 자동으로
- 🏆 **결과 저장**: 라운딩 기록 보관 및 공유
- 👥 **월례회 관리**: 정기 모임 관리

---

## 🚀 시작하기

### 1️⃣ 회원가입
1. 앱 실행 후 **"회원가입"** 버튼 터치
2. 필수 정보 입력:
   - **아이디**: 영문/숫자 조합 (중복 확인 자동)
   - **비밀번호**: 4자리 이상
   - **이름**: 실명 권장
   - **전화번호**: SMS 초대용 (선택)
   - **이메일**: 알림용 (선택)

### 2️⃣ 로그인
- 가입한 **아이디**와 **비밀번호**로 로그인
- 로그인 성공 시 자동으로 **골프약속 화면**으로 이동

## 🎯 주요 기능

### 📱 메인 화면 구성
로그인 후 **골프약속 화면**이 기본 화면입니다:

- **📋 나의 라운드**: 참가 예정인 약속 목록
- **➕ 새 약속**: 골프 약속 생성
- **👥 월례회**: 정기 모임 관리
- **👤 마이페이지**: 개인 정보 및 설정

## 🗓️ 골프 약속 만들기

### 1️⃣ 새 약속 생성
1. **"➕ 새 약속"** 버튼 터치
2. **기본 정보 입력**:
   - **날짜**: 달력에서 선택
   - **시간**: 티오프 시간 (예: 오전 8:00)
   - **골프장**: 골프장명 입력
   - **코스**: 코스명 입력 (선택)

### 2️⃣ 동반자 초대
1. **"동반자 추가"** 버튼 터치
2. 초대 방법 선택:
   - **📱 연락처에서 선택**: 폰 연락처 불러오기
   - **📝 직접 입력**: 이름과 전화번호 입력

### 3️⃣ 초대장 발송
- **"초대하기"** 버튼 터치
- 동반자들에게 **SMS 초대장** 자동 발송
- 초대장에는 **따당 앱 다운로드 링크** 포함

### 4️⃣ 개인 약속 저장
- 동반자 없이 개인 기록용으로도 저장 가능
- **"개인 약속으로 저장"** 선택

---

## 👥 월례회 관리

### 월례회 만들기
1. **"월례회"** 메뉴 선택
2. **"+ 새 월례회"** 버튼 터치
3. **기본 정보 입력**:
   - **월례회명**: 예) "○○CC 토요회"
   - **골프장**: 정기 라운딩 골프장
   - **시간**: 정기 티오프 시간
   - **회장**: 회장 이름
   - **총무**: 총무 이름 (기본: 본인)
   - **회비**: 월 회비 금액
   - **계좌번호**: 회비 납부 계좌

### 정기 일정 설정
- **매월 둘째 주 토요일** 같은 정기 패턴 설정
- **향후 12개월 약속 자동 생성**
- 회원들에게 자동으로 약속 공유

### 회원 관리
- **회원 초대/제거**
- **회장/총무 권한 관리**
- **회비 현황 확인**

## ⚡ 게임 스코어링

### 게임 시작하기
1. 채팅방에서 **"게임 시작"** 버튼 터치
2. **코스 설정**:
   - 골프장 선택 또는 직접 입력
   - 전반/후반 코스명 입력
   - 홀별 파(Par) 설정

3. **플레이어 설정**:
   - 동반자 자동 입력 (채팅 참가자 기준)
   - 순서 조정 가능

4. **베팅 설정**:
   - **타당**: 1타당 금액 (예: 1,000원)
   - **버디**: 버디당 보너스 (예: 10,000원)
   - **베팅 규칙**: 민판/당홀배판 선택

### 스코어 입력
1. **홀별 스코어 입력**:
   - **3, 4, 5, 6, 7, 8, 9, 10타** 버튼으로 빠른 입력
   - 플레이어별로 터치하여 입력

2. **스코어 확인**:
   - 입력한 스코어 즉시 확인
   - 수정 필요 시 다시 터치

3. **홀 이동**:
   - **"다음 홀"** 버튼으로 진행
   - **"이전 홀"** 버튼으로 수정

## 💰 정산 시스템

### 홀별 정산
각 홀 완료 후 **"정산하기"** 버튼으로 즉시 정산:

#### 정산 규칙
- **타당 정산**: 최대 타수와 최소 타수 차이만큼 계산
- **버디 보너스**: 버디(-1) 달성 시 보너스 지급
- **꼴지 벌금**: 해당 홀 꼴지 추가 벌금

#### 정산 방식
1. **민판**: 1등이 꼴지에게서만 받는 방식
2. **당홀배판**: 전체가 꼴지에게 주는 방식

### 최종 정산
- **18홀 완료 후 최종 정산표**
- 플레이어별 **총 수입/지출** 표시
- **순 손익** 한눈에 확인

### 정산 결과 저장
- 정산 내역 자동 저장
- 나중에 **"지난 라운드"**에서 확인 가능

---

## 📂 지난 라운드 보기

### 저장된 게임 확인
1. **마이페이지** → **"지난 라운드"** 선택
2. 저장된 라운드 목록 확인:
   - **날짜/골프장별** 정렬
   - **함께 친 동반자** 표시
   - **최종 정산 결과** 요약

### 상세 결과 보기
- 특정 라운드 선택 시:
  - **홀별 스코어카드**
  - **정산 내역 상세**
  - **플레이어별 통계**

### 결과 공유
- **스크린샷으로 공유**
- **채팅방에 결과 전송**

## 🔧 문제 해결

### 자주 묻는 질문

#### Q1. 초대장을 받은 사람이 앱을 설치하지 않아도 되나요?
**A**: 초대받은 분도 **따당 앱을 설치**해야 채팅 참여 및 실시간 소통이 가능합니다.

#### Q2. 스코어를 잘못 입력했어요.
**A**: 해당 홀에서 플레이어 스코어를 **다시 터치**하면 수정할 수 있습니다.

#### Q3. 정산이 맞지 않는 것 같아요.
**A**: 
- **베팅 설정**을 다시 확인해보세요
- **홀별 정산 내역**을 차례로 검토해보세요
- 문제 지속 시 **"서버 테스트"** 메뉴에서 문의하세요

#### Q4. 채팅방이 안 보여요.
**A**:
- 인터넷 연결 상태를 확인하세요
- 앱을 완전히 종료 후 다시 실행해보세요
- 로그아웃 후 재로그인 시도해보세요

#### Q5. 월례회 약속이 자동 생성되지 않아요.
**A**: 
- 월례회 **정기 일정 설정**을 확인해주세요
- 회장/총무 권한이 있는지 확인해주세요

### 기술 지원
- **서버 연결 문제**: 마이페이지 → 서버 테스트
- **앱 오류**: 앱 재시작 또는 재설치
- **계정 문제**: 로그아웃 후 재로그인

### 연락처
- **개발팀**: 앱 내 문의 기능 이용
- **버그 신고**: 상황 스크린샷과 함께 신고
